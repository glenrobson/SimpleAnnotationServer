Linking Annotations to the Manifest

Download the annotation list below and then you will need to link the annotation list to the correct Canvas or page. This is done by adding an otherContent section to the relevant canvas in the manifest.

Annotation List

Example otherContent section:

{
    "@id": "#{canvas.id}",
    "@type": "sc:Canvas",
    "images": [
        ...
    ],
    "otherContent": [ {
        "@id": "/annotation/list/#{canvas.shortId}.json",
        "@type": "sc:AnnotationList",
        "label": "My fantastic annotations"
    }]
}
